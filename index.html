example-node

一个简单的电商网站demo，使用了nodejs, express, mongodb, mongoose等功能模块。

实现功能：

  1. 注册

  2. 登录

  3. 添加商品

  4. 加入购物车和结算

使用说明：

  1. 启动mongodb；

  2. 启动项目: 在命令行窗口 cd 到项目目录，输入: node app.js

然后就可以通过 http://localhost:3000 访问了。
